Bir að geçidi üzerinden IPv6 rotasýný kaldýrmak

Genellikle elle yapýlmasýna gerek olmaz. Bu iþlem arabirimlerin kapanýþý sýrasýnda að yapýlandýrma betikleri tarafýndan yapýlýr.

ip kullanarak
Kullaným:

/sbin/ip -6 route del ipv6aðý/önek_uzunluðu via ipv6adres [dev aygýt]

Örnek:

# /sbin/ip -6 route del 2000::/3 via 2001:0db8:0:f101::1

route kullanarak
Kullaným:

/sbin/route -A inet6 del ipv6aðý/önek_uzunluðu [dev aygýt]

Yukarýdaki örnekte eklenen rotanýn kaldýrýlmasý: 

# /sbin/route -A inet6 del 2000::/3 gw 2001:0db8:0:f101::1
