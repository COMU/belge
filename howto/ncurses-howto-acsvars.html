ACS_ deðiþkenleri

Eðer DOS'ta programlama yaptýysanýz geniþletilmiþ karakter kümesi içerisindeki þu güzel görünümlü olanlarý biliyorsunuzdur. Sadece bazý uçbirimlerde gösterilirler. NCURSES'ün box() gibi iþlevleri bu karakterleri kullanýr. Tüm bu karakterler seçimlik karakter kümesi anlamýna gelen ACS (alternative character set) ile baþlar. Yukarýdaki programlarýn bazýlarýnda bu karakterleri kullandýðýmý farketmiþ olabilirsiniz. Aþaðýda tüm karakterleri gösteren bir örnek vardýr.

Örnek 5.18. ACS_ deðiþkenleri örneði

#include <ncurses.h>

int main()
{
  initscr();

  printw("Sol üst köþe         "); addch(ACS_ULCORNER); printw("\n");
  printw("Sol alt köþe         "); addch(ACS_LLCORNER); printw("\n");
  printw("Sað alt köþe         "); addch(ACS_LRCORNER); printw("\n");
  printw("Saðý dönük T         "); addch(ACS_LTEE); printw("\n");
  printw("Sola dönük T         "); addch(ACS_RTEE); printw("\n");
  printw("Yukarý dönük T       "); addch(ACS_BTEE); printw("\n");
  printw("Aþaðý dönük T        "); addch(ACS_TTEE); printw("\n");
  printw("Yata çizgi           "); addch(ACS_HLINE); printw("\n");
  printw("Dikey çizgi          "); addch(ACS_VLINE); printw("\n");
  printw("Büyük artý           "); addch(ACS_PLUS); printw("\n");
  printw("Tarama satýrý 1      "); addch(ACS_S1); printw("\n");
  printw("Tarama satýrý 3      "); addch(ACS_S3); printw("\n");
  printw("Tarama satýrý 7      "); addch(ACS_S7); printw("\n");
  printw("Tarama satýrý 9      "); addch(ACS_S9); printw("\n");
  printw("Eþkenar görtgen      "); addch(ACS_DIAMOND); printw("\n");
  printw("Noktalý onay kutusu  "); addch(ACS_CKBOARD); printw("\n");
  printw("Derece sembolü       "); addch(ACS_DEGREE); printw("\n");
  printw("Artý/Eksi iþareti    "); addch(ACS_PLMINUS); printw("\n");
  printw("Madde imi            "); addch(ACS_BULLET); printw("\n");
  printw("Solu gösteren ok     "); addch(ACS_LARROW); printw("\n");
  printw("Saðý gösteren ok     "); addch(ACS_RARROW); printw("\n");
  printw("Aþaðý gösteren ok    "); addch(ACS_DARROW); printw("\n");
  printw("Yukarý gösteren ok   "); addch(ACS_UARROW); printw("\n");
  printw("Diyez iþareti        "); addch(ACS_BOARD); printw("\n");
  printw("Ýçi boþ kare         "); addch(ACS_LANTERN); printw("\n");
  printw("Ýçi dolu kare blok   "); addch(ACS_BLOCK); printw("\n");
  printw("Küçük/Eþit iþareti   "); addch(ACS_LEQUAL); printw("\n");
  printw("Büyük/Eþit iþareti   "); addch(ACS_GEQUAL); printw("\n");
  printw("Pi                   "); addch(ACS_PI); printw("\n");
  printw("Eþit deðil           "); addch(ACS_NEQUAL); printw("\n");
  printw("UK ound sembolü      "); addch(ACS_STERLING); printw("\n");

  refresh();
  getch();
  endwin();

  return 0;
}
