Endiþe etmeyin, postinst ve prerm dosyalarý yanýnda md5sum dosyasý ilk paketiniz için zorunlu deðildir. Fakat bunlarýn olduðunu gözden kaçýrmayýn, her resmi Debian paketi faydalý sebeplerden dolayý bunlara sahiptir.

prerm ve postinst eski belgeleme dosyalarýný kaldýrmakla ve doc'tan share/doc'a bir sembolik bað eklemekle ilgili gözükmektedir. 

$ cat postinst
#!/bin/sh
set -e
# Automatically added by dh_installdocs
if [ "$1" = "configure" ]; then
  if [ -d /usr/doc -a ! -e /usr/doc/parted -a -d /usr/share/doc/parted ]; then
      ln -sf ../share/doc/parted /usr/doc/parted
  fi
fi
# End automatically added section

$ cat prerm
#!/bin/sh
set -e
# Automatically added by dh_installdocs
if [ \( "$1" = "upgrade" -o "$1" = "remove" \) -a -L /usr/doc/parted ]; then
        rm -f /usr/doc/parted
fi
# End automatically added section
