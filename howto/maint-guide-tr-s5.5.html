docs

Bu dosyada, dh_installdocs'un bizim için geçici bir dizine yükleyeceði belgelendirme dosyalarýnýn isimleri belirtilir.

Öntanýmlý olarak kaynak paketinin kök dizinindeki "BUGS", "README*", "TODO" gibi dosyalarýn isimlerini içerecektir.

gentoo için, birkaç tane de ben ekledim: 

BUGS
CONFIG-CHANGES
CREDITS
ONEWS
README
README.gtkrc
TODO

Bu dosyayý kaldýrabiliriz de, bunun yerine bu dosyalarý rules dosyasýnda dh_installdocs komutuna komut satýrý argümanlarý olarak ekleyebiliriz: 

        dh_installdocs BUGS CONFIG-CHANGES CREDITS ONEWS README \
                       README.gtkrc TODO

Yine de, kaynak dosyanýzda böyle herhangi bir dosyaya ihtiyaç duymayabilirsiniz. Bu durumda onu güvenli bir þekilde kaldýrabilirsiniz. Fakat rules dosyasýndaki dh_installdocs çaðrýsýný kaldýrmayýn, çünkü bu copyright dosyasýnýn ve diðer bazý þeylerin kurulumu için kullanýlýr.
