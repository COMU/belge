Paketi http://packages.debian.org/unstable/mail/greylistd adresinde bulabilirsiniz. Debian kullanýcýlarý APT ile kolayca kurabilir:

greylistd'yi devreye sokmak için, acl_rcpt_to ACL'sinde sonuncu accept deyiminden hemen önceye iki deyim ekleyeceðiz. 

  # Belli bir sunucu/gönderici/alýcý üçlüsü için grilisteleme durumunu
  # belirleyecek "greylistd"'yi devreye sokacaðýz.
  #
  # Grilisteleme iletilerini bir boþ gönderici için yapmýyoruz, çünkü
  # boþ gönderici adresli varlýk doðrulamasý iþimize yaramaz (gerçek
  # göndericinin varlýðýný sýnamak için bir konaða posta gönderemeyiz).
  #
  defer
    message     = $sender_host_address is not yet authorized to deliver mail \
                  from <$sender_address> to <$local_part@$domain>. \
                  Please try later.
    log_message = greylisted.
    domains     = +local_domains : +relay_to_domains
    !senders    = : postmaster@*
    set acl_m9  = $sender_host_address $sender_address $local_part@$domain
    set acl_m9  = ${readsocket{/var/run/greylistd/socket}{$acl_m9}{5s}{}{}}
    condition   = ${if eq {$acl_m9}{grey}{true}{false}}

Hatalý teslimat durum bildirimlerini engellemek için zarf gönderici imlerini kullanmýyorsanýz, bunun benzeri bir deyimi acl_data ACL'sine ayrýca boþ göndericili grilisteleme iletileri olarak ekleyebilirsiniz.

Grilistelemenin amaçlarýna uygun olarak veriyi burada yukarýdakinden biraz farklý olarak kullanýyoruz. Boþ olan $sender_address'e ek olarak, bu noktada ne $local_part ne de $domain tanýmlýdýr. Ancak, $recipients, tüm alýcý adreslerinin virgüllerle ayrýlmýþ bir listesini içerir. Meþru teslimat durum bildirimleri için bu deðiþken sadece bir adres içermelidir.

  # uygulayacaðýz. Bunlarý RCPT TO:'dan sonra grilistelemeye konu
  # etmeyeceðiz, çünkü gönderici varlýk doðrulamalarý yaparken
  # karþý konaklarla olumsuz etkileþime girilebilir.
  #

    message     = $sender_host_address is not yet authorized to send \
                  delivery status reports to <$recipients>. \
                  Please try later.
    log_message = greylisted.
    senders     = : postmaster@*
    set acl_m9  = $sender_host_address $recipients
    set acl_m9  = ${readsocket{/var/run/greylistd/socket}{$acl_m9}{5s}{}{}}
