Çok Deðerli Menüler

O_ONEVALUE seçeneðini kapattýðýmýzda ne olur diye merak ediyor olabilirsiniz. Bunun anlamý birden çok öðe seçebilirsiniz demektir. Bu da bizi REQ_TOGGLE_ITEM isteðine getirmektedir. Bir örnekle görelim:

Örnek 5.27. Çok Deðerli Menü Örneði

#include <curses.h>
#include <menu.h>

#define ARRAY_SIZE(a) (sizeof(a) / sizeof(a[0]))
#define CTRLD   4

char *choices[] = {
  "Choice 1",
  "Choice 2",
  "Choice 3",
  "Choice 4",
  "Choice 5",
  "Choice 6",
  "Choice 7",
  "Exit",
};

int main()
{ ITEM **my_items;
  int c;
  MENU *my_menu;
  int n_choices, i;
  ITEM *cur_item;

  /* Curses kipini ilklendir */
  initscr();
  cbreak();
  noecho();
  keypad(stdscr, TRUE);

  /* Öðeleri ilklendir */
  n_choices = ARRAY_SIZE(choices);
  my_items = (ITEM **)calloc(n_choices + 1, sizeof(ITEM *));
  for(i = 0; i < n_choices; ++i)
    my_items[i] = new_item(choices[i], choices[i]);
  my_items[n_choices] = (ITEM *)NULL;

  my_menu = new_menu((ITEM **)my_items);

  /* Menüyü çok deðerli yap */
  menu_opts_off(my_menu, O_ONEVALUE);

  mvprintw(LINES - 3, 0,
    "<SPACE>'i seçmek veya seçileni iptal için kullanýn.");
  mvprintw(LINES - 2, 0,
    "<ENTER> 'ý o anki seçileni görmek için kullanýn "
    "(Çýkmak için F1)");
  post_menu(my_menu);
  refresh();

  while((c = getch()) != KEY_F(1))
  { switch(c)
    { case KEY_DOWN:
        menu_driver(my_menu, REQ_DOWN_ITEM);
        break;
      case KEY_UP:
        menu_driver(my_menu, REQ_UP_ITEM);
        break;
      case ' ':
        menu_driver(my_menu, REQ_TOGGLE_ITEM);
        break;
      case 10:        /* Enter */
      { char temp[200];
        ITEM **items;

        items = menu_items(my_menu);
        temp[0] = '\0';
        for(i = 0; i < item_count(my_menu); ++i)
          if(item_value(items[i]) == TRUE)
          { strcat(temp, item_name(items[i]));
            strcat(temp, " ");
          }
        move(20, 0);
        clrtoeol();
        mvprintw(20, 0, temp);
        refresh();
      }
      break;
    }
  }

  free_item(my_items[0]);
  free_item(my_items[1]);
  free_menu(my_menu);
  endwin();
}

Çok deðerli bir menüde kullanýcý birden fazla öðe seçebilmeli veya seçimi iptal edebilmelidir. REQ_TOGGLE_ITEM isteði o anki seçimi deðiþtirir (etkinleþtirme veya iptal etme). Bu durumda space tuþuna basýldýðýnda REQ_TOGGLE_ITEM isteði menu_driver'a sonucu gerçekleþtirmek için gönderilir.

Ýþte þimdi kullanýcý <ENTER>'a bastýðýnda onun seçtiði öðeleri görürüz. Önce menü ile iliþiklendirilmiþ öðeleri menu_items() iþlevini kullanarak bulduk. Daha sonra öðenin seçili mi deðil mi olduðunu anlamak için öðeler üzerinde gezindik. Eðer öðe seçiliyse item_value() iþlevi TRUE döndürür. item_count() iþlevi menüdeki öðe sayýsýný döndürür. Ayrýca öðe açýklamalarýný item_description() ile alabilirsiniz.
