Çekirdeðin Derlenmesi

Sýradaki adým TimeSys Linux'u derlemektir: 

# make dep
# make bzImage[12]
# make modules
# make modules_install
# make install

make install komutu ile çekirdeðiniz /boot dizinine kopyalanmalý ve hatta GRUB'a TimeSys girdisi eklenmelidir. Eðer yeni çekirdeðinizin öntanýmlý olarak kullanýlmasýný isterseniz /etc/grub.conf dosyasýný düzenleyerek default satýrýný TimeSys çekirdeðinize karþýlýk gelecek þekilde deðiþtirin.


[12] Eðer i586-linux-gcc bulunamadý hatalarý alýrsanýz, Makefile dosyasýný düzenleyerek CROSS_COMPILE ile ilgili bölümü kaldýrýn.
