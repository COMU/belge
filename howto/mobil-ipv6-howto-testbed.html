make install komutunu çalýþtýrdýðýnýzda ne init betikleri ne de yapýlandýrma dosyalarý yüklenmeyecektir. Örnek yapýlandýrma dosyalarý kaynak kodun extra/ dizininde bulunabilir. mip6d.conf(5) kýlavuz sayfasý da yardýmcý olabilir. Dosya, içeriðini kendi açýklayan türdedir.

#
# filename: /etc/mip6d.conf

## If set to > 0, will not detach from tty

##
## IPsec configuration
##

        HomeAgentAddress fec0:106:2700::2;
        HomeAddress fec0:106:2700::2/64;

        IPsecPolicy HomeRegBinding UseESP;
        IPsecPolicy MobPfxDisc UseESP;
        IPsecPolicy TunnelMh UseESP;
  }
