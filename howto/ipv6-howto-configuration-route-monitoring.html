Varolan IPv6 rotalarýnýn görüntülenmesi

Ýlk olarak hangi IPv6 adreslerinin yapýlandýrýlmýþ olduklarý kontrol edilmelidir (belki de otomatik yapýlandýrýlma ile yapýlandýrýlmýþlardýr).

ip kullanarak
Kullaným:

/sbin/ip -6 route show [dev aygýt]

Örnek:

# /sbin/ip -6 route show dev eth0
3ffe:ffff:0:f101::/64 proto kernel metric 256 mtu 1500 advmss 1440
fe80::/10             proto kernel metric 256 mtu 1500 advmss 1440
ff00::/8              proto kernel metric 256 mtu 1500 advmss 1440
default               proto kernel metric 256 mtu 1500 advmss 1440

route kullanarak
Kullaným:

/sbin/route -A inet6

Örnek (çýktý sadece eth0 arabirimini gösterecek þekilde filtrelenmiþtir). Bir arabirim üzerindeki farklý adresler için farklý IPv6 rotalarý görmektesiniz. 

# /sbin/route -A inet6 |grep -w "eth0"
3ffe:ffff:0:f101 ::/64 :: UA  256 0 0 eth0 <== küresel adres için arabirim rotasý
fe80::/10        ::       UA  256 0 0 eth0 <== yerel adres için arabirim rotasý
ff00::/8         ::       UA  256 0 0 eth0 <== tüm çoða gönderim adresleri için arabirim rotasý
::/0             ::       UDA 256 0 0 eth0 <== öntanýmlý rota
