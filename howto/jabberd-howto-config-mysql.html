Yapýlandýrma

Jabberd dosyalarýnýn içerdiði bilgiler arasýnda parolalar da olduðu için gerekli izinleri aþaðýdaki gibi ayarlayalým: 

chown -R root:jabber /usr/local/etc/jabberd/*
chmod -R 640 /usr/local/etc/jabberd/*

Böylece artýk yalnýzca jabber kullanýcýsý ve root'un yapýlandýrma dosyalarýný okuyabilmesini ve deðiþtirebilmesini saðladýk.

Ýpucu

Kolaylýk için yapýlandýrma dosyalarýna bir sembolik link atayabilirsiniz:

ln -s /usr/local/etc/jabberd/ /etc/jabberd

Þimdi yapýlandýrma dosyanýzda veritabaný için ufak bir kaç ayar yapmanýz gerekiyor. Öncelikle sm.xml dosyasýnda "Storage database configuration" ile baþlayan satýrda driver etiketleri arasýnda mysql yazdýðýndan emin olun: 

<!-- Storage database configuration -->
       <storage>
         <!-- By default, we use the MySQL driver for all storage -->
         <driver>mysql></driver>

Daha sonra sm.xml dosyasýnda "MySQL driver configuration" olarak etiketlenmiþ sekmede "secret" yerine parolanýzý yazýn: 

          <!-- MySQL driver configuration -->
          <mysql>
            <!-- Database server host and port -->
            <host>localhost</host>
            <port>3306</port>

            <!-- Database name -->
            <dbname>jabberd2</dbname>

            <!-- Database username and password -->
            <user>jabberd2</user>
            <pass>secret</pass>

            <!-- Transaction support. If this is commented out, transactions
                 will be disabled. This might make database accesses faster,
                 but data may be lost if jabberd crashes.

                 This will need to be disabled if you are using a MySQL
                 earlier than v3.23.xx, as transaction support did not appear
                 until this version. -->
            <transactions/>
          </mysql>
Böylece MySQL ile olan iþimiz bitti.
